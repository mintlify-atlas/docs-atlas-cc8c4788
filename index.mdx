MindFlow is a full-stack web platform built with Next.js, NestJS, and PostgreSQL. It uses an EMA (Ecological Momentary Assessment) chatbot to measure a student's mental fatigue on a 1–5 scale, then automatically breaks down difficult tasks into micro-objectives of 25 minutes or less — reducing cognitive friction and making it easy to take the next step.

## How MindFlow Works

<Steps>
  <Step title="Student logs in">
    Register with an email and password. MindFlow issues a signed JWT that grants access to all protected endpoints for 24 hours.
  </Step>
  <Step title="Start an EMA session">
    The EMA chatbot asks for a fatigue score from 1 to 5. This score is persisted with a UTC timestamp and linked to the active session.
  </Step>
  <Step title="Tasks get intelligently adapted">
    If fatigue ≥ 4, the Task Decomposer calls an external LLM to split every active task into 2–7 micro-objectives, each capped at 25 minutes. If fatigue ≤ 3, tasks are shown in their original form.
  </Step>
  <Step title="Track progress on the Dashboard">
    The Dashboard displays active tasks sorted by deadline, pending micro-objectives grouped by parent task, and a time-series chart of the last 30 fatigue scores.
  </Step>
</Steps>
